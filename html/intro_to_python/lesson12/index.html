Comparison Operators and Booleans

For reasons we shall see shortly, sometimes we need to know if a statement is true or false.

Try in the Python shell:

>>> 6 > 4
>>> 6 == 6
>>> 6 != 6
>>> True and True
>>> True or False
>>> not True

All these expressions return a boolean, which is a data type that has a number of special uses. All the boolean operators are listed here.

  1. Try a testing out the operators above with strings instead of numbers.
  2. Try true/false comparisons with these operators: <=, <>
  3. Try to string multiple comparisons together (e.g. not (True and False), 4 > 3 and 3 > 3).
  4. See what happens when you use = instead of ==.
